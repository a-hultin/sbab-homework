# sbab-homework
Homework assigment for SBAB interview.
A spring-boot application

## How to run
<p>Ensure TRAFIKLAB_APIKEY is set in the environment:</p>
<code>%> export TRAFIKLAB_APIKEY=123...789</code>
<p></p>
<p>Start app</p>
<code>%> mvn spring-boot:run</code>
